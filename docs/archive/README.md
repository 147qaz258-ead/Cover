# 文档归档目录

> 本目录用于归档过期但有参考价值的文档。

---

## 目的 Purpose

保留历史文档供追溯，同时保持活跃文档的整洁。

## 适用范围 Scope

- 已废弃的功能文档
- 旧版本的规格说明
- 历史决策记录
- 过期的指南

## 当前状态 Status

**Active** - 持续使用

## 归档规范

### 1. 文件命名

保留原文件名或使用：
```
<原文件名>-archived-YYYYMMDD.md
```

### 2. 归档标记

在文档开头添加归档声明：

```markdown
> [!WARNING]
> **已归档文档**
> 
> 本文档已于 YYYY-MM-DD 归档，内容可能已过期。
> 当前版本请参阅: [新文档链接](../path/to/new-doc.md)
```

### 3. 原位置处理

在原文件位置留下重定向说明或直接删除（如内容已完全被新文档替代）。
